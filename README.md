# SunrinINT Admin Dashabord Frontend

SunrinINT 어드민 페이지 프론트엔드 저장소

## Project Setup

### Clone

`git clone https://github.com/sunrin-int/sunrinint-admin-v1`

### Dependency Setup

`yarn install`

### Run

`yarn run dev`

## Tech Stack

- NextJS 14 (App Router)
- React.js (Main UI Library)
- SASS (Styling)
